/V1/carts/{cartId}/gift-message/{itemId}

/rest/default/V1/carts/{cartId}/gift-message/{itemId}

GET

In: path - Type: integer - Description: The shopping cart ID.

itemId

In: path - Type: integer - Description: The item ID.
